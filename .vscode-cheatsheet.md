# My VSCODE Stuff (~/.vscode-cheatsheet.md)

## Keyboard Shortcuts

| Key           | Command              |
| ------------- | -------------------- |
| `cmd+k cmd+t` | toggle env. name     |
| `cmd+k cmd+n` | select env. name     |
| `cmd+k cmd+c` | select env. content  |
| `cmd+k cmd+k` | close env.           |
| `cmd+k cmd+b` | open mybib.bib       |
| `cmd+k cmd+h` | open this file       |
